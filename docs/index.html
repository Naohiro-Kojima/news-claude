マネジメント
プレイングマネージャー化で管理職機能が空洞化

リクルートワークス研究所の調査では管理職の87.3%がプレイヤー業務を兼務する「プレイングマネジャー」状態。リクルートMSの意識調査（2024年）では管理職の多くが「メンバーのやる気を高めること」を最も難しい業務と回答。プレイヤー業務の増加がメンバー育成・1on1・心理的安全性の確保を圧迫し、組織の自律性低下につながっている。
